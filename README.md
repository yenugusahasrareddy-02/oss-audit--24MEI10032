Project Description

This repository contains a structured audit of the Linux Kernel and five functional shell scripts as part of the OSS NGMC course.

Shell Scripts Overview

System Identity Report (script1.sh): Displays OS distro, kernel version, and uptime. FOSS Package Inspector 
(script2.sh): Checks for installed packages using a case statement. Disk & Permission Auditor 
(script3.sh): Audits system directory sizes and permissions. Log File Analyzer 
(script4.sh): Counts keywords like 'ERROR' in log files. OSS Manifesto Generator
(script5.sh): Interactive script creating a philosophy .txt file. Instructions to Run
